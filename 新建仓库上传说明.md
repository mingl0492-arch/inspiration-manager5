# 新建 GitHub 仓库上传说明

这是完整的 GitHub 仓库根目录文件包。

## 上传方式

1. 在 GitHub 新建一个空仓库。
2. 解压本压缩包。
3. 打开解压后的文件夹。
4. 把里面的所有文件和文件夹直接上传到 GitHub 仓库根目录。

上传后，仓库根目录应该能看到：

- `.github/workflows/build-windows.yml`
- `src/`
- `package.json`
- `README.md`
- `SECURITY.md`
- `.gitignore`

注意：不要把整个 `inspiration-manager-github-complete-v1.0.8` 文件夹作为一个二级文件夹上传。

## 生成 Windows 安装包

上传完成后，进入：

Actions → Build Windows EXE → Run workflow

构建成功后，在运行记录底部下载：

Artifacts → inspiration-manager-windows-installer

解压后运行：

灵感管理安装包-1.0.8-x64.exe

## 本地开发运行

```bash
npm install
npm run start
```

## 本地打包 Windows 安装包

```bash
npm run dist:win
```

安装包会生成在 `release/` 目录。
