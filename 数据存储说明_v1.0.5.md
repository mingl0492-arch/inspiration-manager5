# 数据存储说明 v1.0.8

从 v1.0.8 开始，灵感数据不再默认存放在 Windows 的 `%APPDATA%` 目录。

## 新数据位置

数据会存放在应用程序所在目录下：

```text
应用安装目录/data/inspiration-data.json
```

例如：

```text
D:/Apps/灵感管理/data/inspiration-data.json
```

这样复制整个应用安装目录时，会同时复制软件和数据。

## 迁移到另一台电脑

1. 在旧电脑找到灵感管理的安装目录。
2. 复制整个安装目录。
3. 粘贴到新电脑任意位置。
4. 运行目录里的 `灵感管理.exe`。

只要 `data/inspiration-data.json` 在这个目录里，数据就会跟着过去。

## 旧版本数据自动迁移

v1.0.8 及更早版本的数据位置通常是：

```text
%APPDATA%/灵感管理/inspiration-data.json
```

v1.0.8 第一次启动时，如果安装目录下还没有数据文件，会自动尝试把旧位置的数据复制到：

```text
应用安装目录/data/inspiration-data.json
```

## 重要提醒

建议把软件安装到当前用户有写入权限的位置，例如：

```text
D:/Apps/灵感管理
C:/Users/你的用户名/Apps/灵感管理
```

不建议安装到需要管理员权限的目录，例如：

```text
C:/Program Files
```

如果安装目录不可写，应用可能无法保存数据。
