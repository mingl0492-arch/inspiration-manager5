# 数据安全说明

这个应用按“本地优先、离线运行”的方式设计：

- 应用运行时不向外部服务器发送数据。
- 用户内容保存到本机 Electron `userData` 目录中的 `inspiration-data.json` 文件。
- 渲染进程关闭 Node.js 直接访问能力，使用 `preload.js` 暴露最小化的保存/读取接口。
- 页面设置了 Content Security Policy，默认禁止加载远程资源。
- 富文本编辑器会移除图片、脚本、iframe、video、audio 等非文本内容。

注意：

- 本项目没有内置加密。如果电脑本身被他人访问，JSON 文件仍可能被读取。
- 需要更高安全级别时，建议后续增加本地密码加密、自动备份和数据导出加密功能。
